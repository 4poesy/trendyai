# TrendyAI API Setup Guide

## 🚀 Quick Setup for Image Generation

Your TrendyAI system is working perfectly! However, to enable **AI image generation**, you need to add your **OpenAI API key**.

### Why the 405 Error Occurred

The error you saw (`OpenRouter API error: 405`) happened because:
- **OpenRouter** is for **text generation** (chat completions), not image generation
- **Image generation** requires **OpenAI API** with DALL-E models

### 🔑 How to Get OpenAI API Key

1. **Visit OpenAI**: Go to [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. **Sign up/Login**: Create an account or log in
3. **Create API Key**: Click "Create new secret key"
4. **Copy the Key**: Save the key (starts with `sk-`)

### ⚙️ How to Add Your API Key

1. **Open API Config Dashboard**: Click the **orange gear icon** in the left toolbar
2. **Find OpenAI Section**: Look for "OpenAI API Configuration"
3. **Enter Your Key**: Paste your OpenAI API key
4. **Test Connection**: Click "Test Connection" to verify
5. **Save**: Click "Save Configuration"

**For Image Generation:**
- 🔑 OpenAI API key (needed for DALL-E)
- 🔑 Stability AI key (alternative)

### 🎨 Test Image Generation

Once you add your OpenAI API key:

1. Go to **StudioMode**
2. Type: `"image of a girl holding bottle water"`
3. The system will generate a real AI image!

### 🔒 Security Note

- API keys are stored **locally** in your browser
- They are **never** sent to external servers
- You can clear them anytime from the API Config Dashboard
